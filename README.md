# Churches in India — Guntur Pilot

Mapping the growth of church construction in India using satellite building
detection. Pilot district: undivided Guntur (Guntur + Palnadu + Bapatla, AP).

**Live map:** https://svsashank.github.io/churches-in-india/

Method, run order, costs and caveats: see [PIPELINE.md](PIPELINE.md).
Scripts in [/scripts](scripts). The map at `index.html` renders the search
frame now, and the full dated dataset automatically once
`data/churches.geojson` is committed after the pipeline run.
